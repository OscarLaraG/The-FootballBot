About this Bot:

This Discord Bot allows users to check stats for certain leagues (currently only Premier League), and small talk, such as:

  1. - Highest Goal scorers
  2. - Top Assisters (To be added)
  3. - Best Goalkeeper (To be added)

  3. 3) Ability to see the highest scorer of ALL TIME for the Premier League

Commands Info


How the commands work:


  • - EVERY command will be pre-fixed by a "?", omitting this will not trigger the command.

  • - Commands are case-sensitive. ?TopScorer and ?topscorer will be recognized as 2 DIFFERENT commands

  • - Commands will have an informative name, the user will be able to tell what the command returns

  • - Commands must be spelled correctly (see below for exact spelling).
